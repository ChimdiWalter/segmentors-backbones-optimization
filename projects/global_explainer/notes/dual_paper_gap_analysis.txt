GAP-P2-1: Oracle gap is substantial (0.327 nMAE)
  Location: experiments/exp_v1/outputs/integrated_full_vs_patch_leaves_18p7_v1/
  Severity: MODERATE (honest limitation, already acknowledged)
  Description: The oracle gap of 0.327 nMAE represents the irreducible error
    floor from granularity mismatch between patch-level predictions and leaf-
    level ground truth. This is ~36% of the patchwise nMAE (0.906), meaning
    roughly a third of the remaining error cannot be removed by better models
    within the current patchwise framework.
  Action: No new experiment needed. The manuscript already frames this honestly.
    Ensure the discussion explicitly states:
    (a) The oracle gap is a property of the aggregation scheme, not the model
    (b) Reducing it requires finer-grained ground truth or hierarchical
        aggregation methods
    (c) Despite the gap, patchwise still beats full-leaf by 21.5%
  Effort: Already addressed in manuscript
  Risk: Reviewers may ask for hierarchical aggregation experiments. Prepare a
    rebuttal paragraph explaining this is future work.

GAP-P2-2: Uncertainty calibration is directional, not perfect
  Location: experiments/exp_v1/outputs/selective_semiamortized_fullstudy174_v3/
  Severity: LOW-MODERATE
  Description: The calibration relationship between predicted uncertainty and
    actual error is monotonic (directionally correct) but not perfectly
    calibrated in the statistical sense (expected calibration error not reported).
  Action: Two options:
    (a) PREFERRED: Compute and report ECE (expected calibration error) and
        reliability diagrams. This quantifies how far from perfect calibration.
        Effort: 2-4 hours.
    (b) MINIMUM: Add a sentence noting that formal calibration metrics (ECE,
        reliability diagrams) are left to future work, and the current monotonic
        relationship suffices for the routing policy.
  Effort: See above
  Risk: Reviewers may request ECE. Having the number ready is prudent.

GAP-P2-3: Cross-collection sample size is limited (28 leaves)
  Location: experiments/exp_v1/outputs/selective_semiamortized_patchstudy_leaves_18p7_v1/
  Severity: MODERATE
  Description: The cross-collection evaluation uses only 28 leaves from
    collection 18.7. This is sufficient for a positive signal but insufficient
    for strong generalization claims.
  Action: Two options:
    (a) PREFERRED: Temper generalization claims. Use language like "preliminary
        cross-collection evidence" rather than "robust generalization."
    (b) OPTIONAL: If additional leaf collections are available, run the pipeline
        on 1-2 more collections to strengthen the claim.
  Effort: (a) 15 minutes; (b) 1-3 days depending on data availability
  Risk: Reviewers will likely note the small sample. Honest framing preempts
    this concern.

GAP-P2-4: 31% both-bad failure rate
  Location: experiments/exp_v1/outputs/integrated_full_vs_patch_leaves_18p7_v1/
  Severity: LOW (acknowledged, taxonomized)
  Description: 31% of images fail under both full-leaf and patchwise approaches.
    The failure taxonomy already characterizes these cases, but root-cause
    analysis of why both methods fail on these specific images is limited.
  Action: Add 2-3 sentences in the discussion analyzing common properties of
    both-bad images (e.g., severe occlusion, extreme color variation, very small
    lesion count). If image metadata is available, a simple correlation analysis
    would strengthen this.
  Effort: 1-2 hours
  Risk: Low. Reviewers appreciate honest failure analysis.

PAPER 2 OVERALL ASSESSMENT:
  The manuscript is mature. All gaps are honest limitations that the text
  already acknowledges. The main improvement opportunities are: (1) computing
  ECE for calibration, (2) tempering cross-collection language, and (3) deeper
  failure root-cause analysis. No fundamental new experiments are required.
