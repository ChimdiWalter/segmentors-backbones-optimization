# manuscript_2 — Paper 2: Black-Box Selective Semi-Amortized Inference

Self-contained snapshot of Paper 2 (v2) with PDF, LaTeX source, all
referenced figures, and an `.org` outline that links to the figures with
relative paths.

```
manuscript_2/
├── paper2_v2_final.pdf          ← compiled PDF
├── main_paper2_v2.tex           ← LaTeX source
├── main_paper2_v2.org           ← outline + relative figure links
└── figures/                     ← every figure referenced by the tex
```

The `.org` file mirrors the LaTeX section structure and embeds each figure as
`[[file:figures/<name>]]` so they render in Emacs / Doom / VS Code Org.
